Denne template skal selvfølgelig justeres efter behov...

...men mange af de grundlæggende ting anbefaler vi at man stort set altid bruger:

  • HTML:
  • viewport tag i <head>
  • "wrapper" div
  • comments hvor relevant (fx 'end wrapper')
  • opdeling i header/nav/main/footer
  • andre semantiske elementer hvor relevant (section, article, aside osv.)
  • struktur som muliggør layout med Flexbox (og senere Grids)
